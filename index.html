Welcome!

This is the home page for the creators and contributors of the Starcounter DBMS. If you are looking for the Starcounter homepage, please go to www.starcounter.com.

The GitHub Starcounter project contains source code for Starcounter contributors. The Starcounter DBMS will be made available under a Dual License.

The current version of Starcounter is only available under a Commercial License. However, the upcoming version 2.0 will be free for open source projects, personal projects (software owned by a single individual) and non-commercial projects.
